### 介绍

> 由于 nginx-http-flv-module 官方镜像是四年前的，想弄个最新版的，就自己打了一个包，这个项目仅作为保留使用

1. 本项目使用最新版 `nginx-http-flv-module` 目前版本为 `1.2.10`
2. rtmp 和 http nginx 解析都为 stream
